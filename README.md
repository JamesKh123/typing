# Typing Race (Node.js + Socket.IO) - Completed ZIP

This is a ready-to-deploy Node.js + Socket.IO typing race demo.

## Quick local run
1. Install Node.js (v18 recommended)
2. Install deps:
   ```
   npm install
   ```
3. Run:
   ```
   npm start
   ```
4. Open http://localhost:3000 and create/join a room.

## Deploy to Render (recommended)
1. Push this repo to GitHub
2. Create a new Web Service on Render and connect your repo.
3. Build command: `npm install`
4. Start command: `npm start`
5. Render will set PORT automatically.

## Files included
- server.js
- package.json
- Procfile (optional hosts)
- render.yaml (Render quick-deploy)
- public/index.html, public/client.js
- README.md
